Theorem. There are infinitely many prime numbers.

Proof. Suppose, for contradiction, that there are only finitely many primes p_1,...,p_r. Let N=p_1...p_r+1. Any prime divisor q of N is not equal to any p_i, since division by p_i leaves remainder 1. This contradicts the assumption that p_1,...,p_r list all primes.
