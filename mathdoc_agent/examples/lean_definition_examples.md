Definition. A sorted list over a type alpha with an order relation consists
of a list xs : List alpha together with a proof that xs is sorted.

Class. A magma consists of a type carrier and a binary operation mul on carrier.
The operation is part of the data of the class.

Instance. The natural numbers form a magma using Nat as the carrier and addition
as the binary operation.

Inductive proposition. Even is the proposition on natural numbers generated by
zero_even, saying that zero is even, and step_even, saying that if n is even then
n+2 is even.

Inductive type. A binary tree with labels in alpha has constructors leaf with no
arguments and node with a left tree, a label, and a right tree.

Definition. A bounded natural number consists of a value n : Nat together with a
proof that n is at most a fixed bound b.
