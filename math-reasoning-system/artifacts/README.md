# Mathematical Artifacts

This directory contains persistent mathematical artifacts generated during the reasoning loop.

## Key Artifacts
- `01_problem_formalization.md`: Formalized problem components and definitions.
- `02_literature_review.md`: Insights and techniques from source material.
- `02_examples.md`: Early sanity-check examples, boundary cases, and candidate counterexamples.
- `03_strategy_plans.md`: Proposed proof sketches and difficulty ratings.
- `04_execution_trace.md`: Step-by-step trace of the execution engine.
- `05_final_manuscript.md`: The complete, verified proof or result.
- `05_failure_report.md`: Detailed analysis of why the loop terminated without a solution.

## Exploratory Research Artifacts
- `02_research_questions.md`: Precise questions, conjecture candidates, and ranked research directions.
- `03_exploration_plan.md`: Concrete probes for examples, counterexamples, special cases, analogies, and source checks.
- `04_research_trace.md`: Evidence accumulated during exploratory probes and conjecture stress tests.
- `05_research_brief.md`: Consolidated research output with facts, conjectures, obstacles, and next steps.
- `05_research_obstacles.md`: Failure or pause report for exploratory work.
